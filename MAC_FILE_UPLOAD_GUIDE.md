# 📁 Mac 系统文件上传指南

## ⚠️ 重要提醒：请选择文件，不要选择文件夹！

**常见错误**：在文件选择器中选择了文件夹而不是文件。

**如何识别文件夹**：
- ✅ **文件**：显示文件扩展名，如 `document.pdf`、`report.docx`
- ❌ **文件夹**：显示"X个项目"，如 `相关帮助文档.docx (7个项目)`

**正确操作**：
1. 如果看到显示"X个项目"的项，说明这是文件夹
2. **双击**文件夹进入内部
3. 选择具体的文件（不显示"X个项目"的项）
4. 点击"打开"按钮

---

## 问题说明

在 Mac 系统上，文件选择对话框默认打开在"下载"文件夹。本指南将帮助您导航到其他文件夹。

## 🎯 快速解决方案

### 方法 1：使用下拉菜单切换位置 ⭐推荐

1. 点击文件选择器顶部的**下拉菜单**（显示"下载"的位置）
2. 从列表中选择您需要的位置：
   - 🏠 **最近使用** - 最近访问的文件夹
   - 💻 **桌面** - 桌面文件夹
   - 📄 **文稿** - 文档文件夹
   - 📥 **下载** - 下载文件夹
   - iCloud 云盘
   - 其他位置...

### 方法 2：使用键盘快捷键 ⌨️

**Cmd + Shift + G** - 前往文件夹
1. 在文件选择器中按下 `Cmd + Shift + G`
2. 输入文件夹路径，例如：
   ```
   ~/Desktop        # 桌面
   ~/Documents      # 文稿
   ~/Downloads      # 下载
   /Users/你的用户名/文件夹名称
   ```
3. 按回车键确认

### 方法 3：使用左侧边栏 📂

点击文件选择器左侧边栏中的位置快捷方式：
- 🔵 **最近使用**
- ⭐ **个人收藏**
- 🔵 **应用程序**
- 💾 **桌面**
- 📄 **文稿**
- 📥 **下载**
- iCloud 云盘

### 方法 4：拖放文件 🖱️

虽然文件选择器不支持拖放，但您可以：
1. 先将文件复制到"下载"文件夹
2. 然后在文件选择器中选择

## 🔍 详细步骤（图文说明）

### 步骤 1：打开文件选择器
点击知识库管理中的"上传文档"按钮。

### 步骤 2：导航到目标文件夹

**选项 A：使用顶部下拉菜单**
```
┌─────────────────────────────────────┐
│  [下载 ▼]  [视图] [排序]            │  ← 点击这里
├─────────────────────────────────────┤
│  最近使用                           │
│  桌面                              │
│  文稿                              │
│  下载                    ✓         │
│  iCloud 云盘                        │
└─────────────────────────────────────┘
```

**选项 B：使用键盘快捷键**
```
1. 按 Cmd + Shift + G
2. 输入路径：~/Desktop
3. 按回车
```

### 步骤 3：选择文件
1. 找到您的文件
2. 可以按住 `Cmd` 键多选文件
3. 点击"打开"按钮

## 📋 支持的文件格式

- ✅ **PDF** - Portable Document Format (.pdf)
- ✅ **DOC** - Microsoft Word 97-2003 (.doc)
- ✅ **DOCX** - Microsoft Word (.docx)
- ✅ **MD** - Markdown (.md)
- ✅ **TXT** - 纯文本 (.txt)

## ⚠️ 文件要求

- 📏 **最大文件大小**：50MB
- 📊 **文件数量**：支持一次选择多个文件
- 🔒 **安全性**：所有文件将加密存储在 Supabase Storage

## 💡 实用技巧

### 快速访问常用文件夹

1. **添加到边栏**：
   - 在访达中找到常用文件夹
   - 拖动到边栏的"个人收藏"部分
   - 以后文件选择器中会显示

2. **使用搜索功能**：
   - 点击文件选择器右上角的搜索框
   - 输入文件名搜索

3. **按文件类型过滤**：
   - 文件选择器已自动过滤，只显示支持的格式

### 批量上传文件

1. 按住 `Cmd` 键
2. 点击多个文件进行选择
3. 点击"打开"一次性上传所有选中的文件

## 🐛 常见问题

### Q1: 为什么只能看到"下载"文件夹？
**A**: 这是默认位置。使用上述方法切换到其他文件夹。

### Q2: 文件夹显示为灰色，无法访问
**A**: 可能是权限问题。请确保：
- 您有该文件夹的读取权限
- 文件夹不是系统保护文件夹

### Q3: 找不到我的文件
**A**: 尝试：
1. 使用搜索功能
2. 按 Cmd+Shift+G 输入完整路径
3. 检查文件格式是否支持

### Q4: 上传时显示"格式不支持"
**A**: 确认文件扩展名为：.pdf, .doc, .docx, .md, 或 .txt

## 🎬 操作视频提示

如果您不熟悉 Mac 文件选择器，建议：
1. 使用 **Cmd + Shift + G** 是最快的方法
2. 或者先将文件复制到"下载"文件夹
3. 可以在访达中查看文件的完整路径：
   - 右键点击文件
   - 按住 `Option` 键
   - 选择"拷贝'xxx'的路径名称"
