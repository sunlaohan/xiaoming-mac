# 🚀 快速上传参考卡

## 📌 一分钟速查

### ❓ 如何判断选择的是什么

| 类型 | 显示特征 | 图标 | 操作 |
|------|---------|------|------|
| ✅ **文件** | 文件名.pdf<br/>2.5 MB | 📄 | 选中 → 点击"打开" |
| ❌ **文件夹** | 文件夹名<br/>7个项目 | 📁 | 双击进入内部 |

---

## 🎯 三步上传法

```
第1步：点击【上传文档】
  ↓
第2步：导航到正确的文件夹
  • 使用顶部下拉菜单
  • 或按 Cmd+Shift+G 输入路径
  ↓
第3步：选择文件（不是文件夹）
  • 单击选择一个文件
  • Cmd+点击 = 多选
  • Cmd+A = 全选
  ↓
第4步：点击【打开】按钮
```

---

## ⚠️ 常见错误

### 错误 #1：选择了文件夹

**症状**：
```
相关帮助文档.docx
7个项目              ← 看到这个就是文件夹！
```

**正确做法**：双击进入文件夹，选择里面的文件。

---

### 错误 #2：弹窗关闭

**原因**：
- 选择了文件夹而不是文件
- 误点击了弹窗外部

**解决**：重新打开，确保选择文件。

---

### 错误 #3：找不到文件

**解决方案**：

| 问题 | 解决方法 |
|------|---------|
| 默认在"下载"文件夹 | 点击顶部下拉菜单切换 |
| 文件在其他位置 | 按 Cmd+Shift+G 输入路径 |
| 文件名记不清 | 使用右上角搜索框 |

---

## 🔧 Mac 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd + Shift + G` | 前往文件夹（输入路径） |
| `Cmd + 点击` | 多选文件 |
| `Cmd + A` | 全选当前文件夹中的文件 |
| `回车` | 打开选中的文件夹 |
| `空格` | 快速预览文件 |

---

## 📋 上传前检查清单

点击"打开"之前，确认：

- [ ] 选中的是**文件**，不是文件夹
- [ ] 文件显示扩展名（.pdf, .docx, .txt, .md）
- [ ] **没有**显示"X个项目"
- [ ] 文件大小 ≤ 50MB
- [ ] 文件格式是支持的类型

**全部勾选？** 可以点击"打开"了！

## 💡 专业技巧

### 快速导航

**方法 1：下拉菜单**
```
点击顶部 [下载 ▼] → 选择目标文件夹
```

**方法 2：快捷键**
```
Cmd+Shift+G → 输入 ~/Desktop → 回车
```

**方法 3：侧边栏**
```
点击左侧 [桌面] [文稿] [下载]
```

---

### 批量上传

1. 进入包含文件的文件夹
2. 按住 `Cmd` 键
3. 点击多个文件
4. 点击"打开"
5. 所有文件将依次上传

---

### 常用路径

| 位置 | 路径 |
|------|------|
| 桌面 | `~/Desktop` |
| 文稿 | `~/Documents` |
| 下载 | `~/Downloads` |
| 自定义 | `/Users/用户名/文件夹` |

使用 `Cmd+Shift+G` 输入这些路径即可快速跳转。

## 🎨 可视化指南

### 文件 vs 文件夹

**文件（正确）**：
```
┌──────────────────────┐
│ 📄 用户手册.pdf      │  ← 选择这个
│    2.5 MB           │
└──────────────────────┘
```

**文件夹（错误）**：
```
┌──────────────────────┐
│ 📁 帮助文档          │  ← 双击进入
│    7个项目           │     不要直接打开
└──────────────────────┘
```

## 🆘 故障排除

### 问题：上传后没有反应

**检查**：
1. 文件格式是否支持？
2. 文件大小是否超过 50MB？
3. 是否选择了文件而不是文件夹？

**解决**：
- 查看页面顶部的提示消息
- 检查浏览器控制台（F12）
- 重新尝试上传

---

### 问题：找不到文件选择器

**可能原因**：
- 弹窗被浏览器阻止
- 文件选择器在其他桌面空间

**解决**：
- 检查浏览器设置，允许弹窗
- 按 `Cmd+Tab` 切换窗口
- 关闭知识库弹窗，重新打开

---

## 📞 快速帮助

### 支持的文件格式

| 格式 | 扩展名 | 最大大小 |
|------|--------|---------|
| PDF | .pdf | 50MB |
| Word 97-2003 | .doc | 50MB |
| Word | .docx | 50MB |
| Markdown | .md | 50MB |
| 文本 | .txt | 50MB |

### 不支持的格式

❌ 图片文件（.jpg, .png）  
❌ 压缩文件（.zip, .rar）  
❌ 可执行文件（.exe, .app）  
❌ 视频文件（.mp4, .mov）  
❌ 文件夹
