# 小鸣同学

小鸣同学是一款支持知识库问答的桌面 AI 助手。

## 下载与安装

请选择与你的 Mac 芯片匹配的安装包：

| Mac 芯片 | 安装包 | 适用设备 |
| --- | --- | --- |
| Apple 芯片 | [下载 arm64 版本](https://github.com/sunlaohan/xiaoming-mac/releases/download/v1.0.0/%E5%B0%8F%E9%B8%A3%E5%90%8C%E5%AD%A6-1.0.0-arm64.dmg) | M1、M2、M3、M4 及后续 Apple 芯片 Mac |
| Intel 芯片 | [下载 x64 版本](https://github.com/sunlaohan/xiaoming-mac/releases/download/v1.0.0/%E5%B0%8F%E9%B8%A3%E5%90%8C%E5%AD%A6-1.0.0-x64.dmg) | Intel 处理器 Mac |

下载后打开 DMG，将“小鸣同学”拖入 `Applications` 文件夹即可安装。首次打开时，如 macOS 显示安全提示，可在 Finder 中右键点击应用并选择“打开”。

## 本地运行

```bash
npm install
npm run dev
```
